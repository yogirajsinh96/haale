haale is a simple self-hosted CI/CD platform for GitHub. Deploy apps on your VPS without complex DevOps.

Install haale Learn More

Features

GitHub Integration

Connect your GitHub repository and deploy automatically on push.

Docker Powered

Build and run containers automatically on your VPS using Docker.

Live Deployment Logs

See real-time logs during build and deployment.

How It Works

1. Install

Install haale on your VPS using npm or npx.

2. Connect

Connect your GitHub repository and configure your project.

3. Deploy

Push code and watch it go live automatically.

Install haale

Get started in seconds. Requires Node.js and Docker installed.

npm install -g haale
haale start
